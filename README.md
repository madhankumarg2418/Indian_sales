# 📊 Indian Sales Dashboard

A Power BI dashboard for analyzing sales performance across India. The project provides interactive visualizations and key insights into revenue, profit, and regional trends to support data-driven decisions.

## 🚀 Features
- Sales and profit analysis
- Regional performance tracking
- Interactive charts and filters
- KPI monitoring and trend analysis

## 🛠️ Tools Used
- Power BI
- Microsoft Excel
- Power Query
- DAX

## 📂 Files
- `Indian_Sales_Dashboard.pbix`
- `Indian_Sales_Data.xlsx`

## 📈 Purpose
This project demonstrates how business intelligence tools can transform raw sales data into meaningful insights and help monitor overall sales performance.
